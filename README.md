# Policy Navigator

A smart CLI-based Policy Management System built using Python and Object-Oriented Programming that helps citizens and authorities efficiently manage, browse, and navigate government policies.

---

## Features

### Public User Features
- View policies by category
- View all available policies
- Browse policy details
- Submit suggestions and feedback

### Authority Features
- Secure authority login
- Add new policies
- Edit existing policies
- Remove policies
- View citizen suggestions

## Workflow

### Public User Workflow

1. Select PUBLIC mode
2. Browse policies by category
3. View policy details
4. Submit suggestions

### Authority Workflow

1. Select AUTHORITY mode
2. Enter authority password
3. Manage policies
4. Review public suggestions
